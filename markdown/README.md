# Markdown - Getting started

## Headings

* \# text - h1
* \#\# text - h2
* \#\#\# text - h3
* \#\#\#\# text - h4
* \#\#\#\#\# text - h5
* \#\#\#\#\#\# text - h6

## Paragraph

Type as per as it is no need for any thing.<br>
use \<br\> for break the paragraph

## Italic

- \*text\* - *text*
- \_text\_ - _text_

## Bold

- \*\*text\*\* - **text**
- \_\_text\_\_ - __text__

## Bold and Italic

- \*\*\*text\*\*\* - ***text***
- \_\_\_text\_\_\_ - ___text___

## Block quotes

- \> text 
> text
- \>\> text
>> text

## Ordered list

\1\. text<br>
\2\. text<br>
1. text
2. text
	1. text
	2. text

## Unordered list
\- text<br>
\- text<br>
- text
- text
	- text
	- text

## code block
\`\`\`<br>
code here<br>
\`\`\`<br>

(e.g)
```bash
cd ..
```

## Links
<span>\[</span>Content here<span>](</span>link here)<br>
[Google](https://www.google.com)

## Images
<span>\!</span><span>\[</span>Title here<span>](</span>path here)<br>
